Dark Sky Finder

This is a prototype for a website that helps users plan stargazing outings and connect with other amateur astronomers.

I designed all aspects of the site, using Sketch for wireframing and asset creation, and InVision for the interactive prototype.

Includes UI for light pollution map, weather, planetary rise and set times, and social aspects.

5e Character Generator

This web app does the heavy lifting for users in creating a level 1 player character in a tabletop role-playing game like Dungeons & Dragons.

I wrote much of the business logic and UI logic, created the logo, and made the page mobile responsive. I'm currently working on supporting more classes and printing stats directly onto a character sheet.
